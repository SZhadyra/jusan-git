# stepik-id-printer (h1)

Это проект для вывода моего персонального идентификатора на _образовательной платформе_ `stepik.org`.

## Возможности (h2)

Данный проект имеет следующий функционал.

- Выводит в **stdout** уникальный идентификатор.
- Без зависимостей.
- Простой.

## Запуск (h2)

```bash
bash ./script.sh
```
